# Medicine Ordering System

The Medicine Ordering System is a database-driven academic project developed using Oracle SQL Developer and Oracle Forms Builder.  
The system is designed to manage medicines, customers, and orders efficiently through a form-based application connected to an Oracle database.

---

## Project Overview

This project demonstrates the implementation of a structured database system integrated with Oracle Forms.  
It allows users to manage medicine inventory, maintain customer records, and process medicine orders in a simple and organized manner.

---

## Technologies Used

- Oracle SQL Developer  
- Oracle Database  
- SQL  
- PL/SQL  
- Oracle Forms Builder  

---

## System Features

- User login authentication  
- Medicine stock management  
- Customer details management  
- Medicine ordering process  
- Database-backed order storage  
- Form-based graphical user interface  

---

## System Modules

- Login Module  
- Medicine Entry Module  
- Customer Management Module  
- Order Processing Module  
- Report / Output Module  

---

## Database Design

The database consists of multiple related tables to store:
- Medicine information  
- Customer details  
- Order records  

Primary keys, foreign keys, and constraints are used to ensure data integrity and consistency.

---

## How to Run the Project

1. Open **Oracle SQL Developer**.
2. Execute the SQL script provided in the `Database` folder to create the required tables.
3. Open **Oracle Forms Builder**.
4. Load the provided `.fmx` file.
5. Connect the form to the Oracle database.
6. Run the application.
